  LOGIN PAGE  (/login)
  ═══════════════════════════════════════════════════════
  - Split layout: left side is brand panel (dark navy, orange logo, tagline)
  - Right side: centered login form
  - Fields: Email, Password (show/hide toggle)
  - Forgot password link → shows email input modal → sends reset email
  - Validates with React Hook Form + Zod
  - On success: stores accessToken in memory (Zustand), redirects to /dashboard
  - Auto-refresh token on 401 using axios interceptor

  ═══════════════════════════════════════════════════════
  REUSABLE COMPONENTS — build all of these
  ═══════════════════════════════════════════════════════
  DataTable: wraps TanStack Table with:
    - Search input, column visibility toggle, export CSV button
    - Pagination: prev/next, page size selector (10/20/50)
    - Loading skeleton rows, empty state illustration
    - Row click handler prop, checkbox selection
  
  StatCard: displays a KPI metric
    - Icon (colored), title, value, change% (green up / red down arrow)
    - Optional mini sparkline chart (Recharts)
  
  PageHeader: top of each page
    - Title, subtitle, breadcrumb, action button slot
  
  StatusBadge: colored pill for any status
    - Accepts: status string → maps to color automatically
    - OPEN=blue, RESOLVED=green, BREACH=red, PENDING=yellow, etc.
  
  FormModal: slide-over panel from right (not a centered modal)
    - Title, close button, form slot, footer with Cancel/Save buttons
    - Loading state on submit button
  
  ConfirmDialog: standard shadcn AlertDialog for delete confirmations
  
  FileUpload: drag-and-drop zone
    - Shows file preview for images, name+size for docs
    - Uploads to Supabase Storage on select, returns URL
    - Accepts file types and max size as props
  
  Avatar: initials fallback if no photo URL
  
  EmptyState: illustration + heading + subtext + optional CTA button
